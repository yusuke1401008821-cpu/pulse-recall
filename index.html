Pulse Recall とは

医学部試験対策と USMLE 医学英語に寄せた学習アプリ

Pulse Recall は、医学部の授業試験、学内の過去問対策、USMLE Step 1 を中心とした医学英語の暗記を、 `過去問 → 関連講義 / スライド → 理解 → 暗記 → 演習` の流れで進めるための Web アプリです。

医学部の授業試験対策

過去問参照、講義 PDF 取り込み、理解モード、暗記モードを同じアプリ内でつなげて、定期試験や CBT 前の復習を組み立てやすくしています。

USMLE 医学英語の基礎固め

`医学単語入門` や身体部位デッキを起点に、接頭辞・接尾辞、解剖用語、医学英語の基本語彙を理解してから暗記へ進めます。

理解してから暗記する流れ

初学者向けの理解モードで内容を読み、理解後に暗記と演習へつなげる設計なので、知識ゼロからでも入りやすくしています。

医学英語・過去問・画像カード対応

医学英単語、講義スライド、過去問 PDF、画像付きカード、4択や記述演習まで一つの学習導線で扱えます。

ようこそ

はじめての 3 ステップ

1. 身体部位から始める

まずは `身体部位の名称` で、解剖の基本語を理解モードから読み進めると入りやすくなります。

2. 医学単語を理解する

`医学単語入門` で接頭辞・接尾辞を理解しておくと、授業試験と USMLE の語彙を広げやすくなります。

3. 過去問から広げる

`過去問参照` と `PDF取り込み` を使うと、授業試験や USMLE 向けの理解と暗記を同じ流れで広げられます。
